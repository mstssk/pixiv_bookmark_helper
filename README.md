# Pixiv Bookmark Helper
pixivのブックマーク機能を補助します。

* pixivのブックマーク編集画面にタグを一括登録するボタンを追加
* 拡張機能のコンテキストメニューにブックマーク一覧へのリンクを表示

ユーザースクリプト版: https://gist.github.com/mstssk/913bfa9b394cd6fa791919478454497d/
